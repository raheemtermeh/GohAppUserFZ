## Funzone App (React + Vite + Tailwind)

Ready-to-install mobile-style event reservation app scaffold.

### Install

```bash
npm i
```

### Dev

```bash
npm run dev
```

### Build

```bash
npm run build && npm run preview
```

### Features

- Home page with event list and filter chips
- Filters page (venue, price, date)
- Event details and reservation flow
- Login and Profile pages with mock state
- Bottom tab navigation
